# 销售赋能（Sales Enablement）

## 功能描述

为销售团队生成专业、即用的销售工具包。涵盖六大核心销售材料类型：

1. **推销演示文稿（Pitch Deck）**：10-12 页投资级演示文稿大纲，包含痛点分析、解决方案、产品演示、案例展示和行动号召
2. **一页纸摘要（One-Pager）**：单页产品摘要，适合邮件发送和展会分发
3. **异议处理手册（Objection Handling）**：常见客户异议及应对策略，含话术模板
4. **演示脚本（Demo Script）**：20-30 分钟产品演示脚本，基于"情境-问题-方案-效果"叙事框架
5. **ROI 计算器（ROI Calculator）**：投资回报分析模型，含关键假设和计算公式
6. **竞争战卡（Battle Card）**：与竞品的逐项对比，突显差异化优势

适用于 B2B 销售场景，帮助销售团队快速建立信任、传递价值、应对异议、推动成交。

## 使用示例

**示例 1：生成 ROI 计算器**
```
/销售工具
产品名称：SmartWash X9 全自动洗车机
工具类型：roi
目标客户画像：中型加油站（日均洗车量 50-100 台）
```

**示例 2：生成完整销售工具包**
```
/sales-tools
产品名称：云洗车管理 SaaS
工具类型：all
目标客户画像：拥有 3-10 家门店的连锁洗车品牌
竞品信息：竞品 A 价格较低但功能少；竞品 B 功能全但价格高
```

## 执行流程

### 第一步：产品价值梳理
分析产品核心功能、差异化优势、目标客户痛点，提炼 3 个核心价值主张。

### 第二步：客户画像分析
基于目标客户画像，分析其决策链、预算权限、采购周期和核心关注点。

### 第三步：工具内容生成
根据指定的工具类型，按照对应模板生成结构化内容。每类工具有独立的框架和最佳实践。

### 第四步：话术优化
为关键场景（价格谈判、竞品对比、试用转化）生成销售话术，包含破冰、提问、呈现、促成四个环节。

### 第五步：交付格式化
将生成内容以清晰可用的 Markdown 格式输出，方便直接导入飞书文档或 Notion。
