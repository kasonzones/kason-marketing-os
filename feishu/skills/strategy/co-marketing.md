# 联合营销（Co-Marketing）

## 功能描述

为企业寻找潜在合作伙伴并设计联合营销活动。覆盖五种合作类型：

1. **互补产品合作（Complementary）**：与产品互补但非竞争的品牌合作（如自动洗车机 + 车用清洁剂品牌）
2. **技术集成营销（Tech Integration）**：与技术平台/软件开发联合营销（如 SaaS 产品的 API 集成合作伙伴）
3. **渠道合作（Channel）**：与经销商、代理商、系统集成商联合推广
4. **内容合作（Content）**：联合发布行业报告、白皮书、线上研讨会
5. **活动合作（Event）**：联合参展、联合举办客户沙龙、联合直播

核心理念：1+1>2 的乘法效应——找对你的"另一半"，让你的营销资源翻倍。联合营销的本质是"用对方已有的信任资产，为你的品牌背书"。

## 使用示例

**示例 1：互补产品合作**
```
/联合营销
产品名称：SmartWash X9 全自动洗车机
合作伙伴类型：complementary
行业：汽车后市场
营销目标：lead-gen
```

**示例 2：技术集成营销**
```
/co-marketing
产品名称：CloudCarWash SaaS
合作伙伴类型：tech-integration
行业：汽车服务行业 SaaS
营销目标：product-launch
```

## 执行流程

### 第一步：合作伙伴画像
定义理想合作伙伴的特征：产品互补性、客户群重叠度、品牌调性匹配度、合作意愿评估。

### 第二步：潜在合作方挖掘
生成潜在合作方搜索策略：行业展会名单、社交媒体共同关注者分析、API 市场的互补应用、客户使用链分析。

### 第三步：联合营销方案设计
基于合作类型和营销目标，设计 1-2 个具体联合营销活动方案，包含：活动形式、双方分工、内容规划、推广渠道、时间安排。

### 第四步：合作提案生成
撰写面向潜在合作伙伴的商务提案，包含合作价值阐述（Why This、Why Now、Why Us + You）、合作模式建议、资源投入预估和预期收益。

### 第五步：效果衡量框架
定义联合营销活动的关键指标（联合获客数、品牌曝光量、交叉转化率、合作 ROI），建立数据共享和效果评估机制。

期待回复，
{姓名} | 凯森智能科技

```
## 📊 效果衡量指标

| 指标 | 定义 | 目标 |
|------|------|------|
| 联合新客数 | 通过对方渠道首次接触我方的客户数 | > 500 |
| 交叉转化率 | 接触一方后转化为另一方客户的比例 | > 5% |
| 联合活动曝光量 | 双方渠道总曝光量 | > 50 万 |
| 合作 ROI | (联合获客价值 - 合作投入) / 合作投入 | > 300% |
```

## 关联技能

- `/客户挖掘` — 合作伙伴的客户群可作为挖掘来源
- `/公共关系` — 联合新闻发布和媒体传播
- `/发布计划` — 联合发布会/发布活动策划
- `/市场营销创意` — 联合营销的创意方案
- `/销售赋能` — 联合销售材料制作

## 配置要求

- 需明确合作伙伴类型和营销目标
- 建议提供可共享的品牌资产清单（内容、渠道、用户量等）
- 合作提案前建议调研对方近期的营销动态和品牌方向
