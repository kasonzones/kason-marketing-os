# 公共关系（Public Relations）

## 功能描述

为品牌提供专业的公共关系策略和媒体沟通方案。覆盖四大 PR 核心场景：

1. **记者联络（Pitch）**：撰写媒体联络邮件，生成定制化 Pitch 模板，使你的新闻稿在海量投稿中脱颖而出
2. **热点借势（Newsjacking）**：实时捕捉行业热点，快速生成关联评论和内容，将品牌自然植入公共对话
3. **媒体资料包（Press Kit）**：构建完整的媒体资料包，包含品牌故事、核心数据、高清素材、发言人简介和 FAQ
4. **危机公关（Crisis）**：提供危机评估、应对框架和沟通策略，帮助品牌在负面事件中维护声誉

核心理念：帮助记者完成他们的工作，而非单纯推销你的产品。好的 PR 是"给记者一个值得写的故事"。

## 使用示例

**示例 1：产品发布媒体联络**
```
/公关
公司名称：凯森智能科技
主题：SmartWash X9 全自动洗车机正式发布 — "重新定义中国加油站洗车体验"
类型：pitch
目标媒体：科技媒体、加油站行业媒体、地方商业媒体
```

**示例 2：热点借势**

```
/pr

```

## 执行流程

### 第一步：新闻价值评估
分析主题的新闻价值：是否有时效性、冲突性、影响力、人情味和接近性。评估目标媒体是否会对该话题感兴趣。

### 第二步：故事角度设计
为同一事件设计 3-5 个不同的报道角度（行业趋势角度、用户故事角度、技术创新角度、商业数据角度、社会影响角度），匹配不同媒体类型的偏好。

### 第三步：内容生成
根据 PR 类型生成对应内容：记者联络邮件（含 subject line、正文、follow-up 策略）、热点借势评论、媒体资料包组件。

### 第四步：发布时机与渠道规划
分析最佳发布时间窗口（避开重大新闻日、利用行业会议时间点），推荐媒体分发渠道（通讯社、垂直媒体、综合媒体）和发布节奏。

**Subject**: 3分钟洗完一辆车？这家公司想重新定义中国加油站的洗车体验

**正文**：
```
{记者姓名} 老师你好，

## 🎙️ 热点借势：加油站排队危机

**事件背景**：近期多地加油站因洗车排队引发交通拥堵，城管部门介入整治。

**Pitch 角度**：
> "城管在整治加油站排队，但治标不治本——问题的根源不是'排队管理'，而是'洗车效率'。3分钟全自动洗车，从源头解决排队问题。"

## 📦 媒体资料包清单

- [ ] 品牌故事（1页）：创始人故事、公司愿景
- [ ] 产品介绍 PDF：核心参数、功能亮点
- [ ] 核心数据一页：用户量、洗车量、满意度
- [ ] 高清素材包：产品图、场景图、CEO 头像（带外链）
- [ ] 客户案例速览：3个标杆客户一句话介绍 + 数据
- [ ] 发言人简介：CEO/CTO 个人背景和可采访话题
- [ ] FAQ 文档：媒体常见问题 15 问 + 标准回答
```
