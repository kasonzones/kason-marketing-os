# 客户挖掘（B2B Prospecting）

## 功能描述

为 B2B 产品系统化地定义理想客户画像（ICP）、构建潜在客户清单、建立资格评估和评分体系。帮助销售和市场团队从模糊的"目标客户"转向精确的"可触达客户"。

核心能力：
- **ICP 定义**：从行业、规模、现状、痛点、决策链五个维度精准画像
- **客户列表构建**：提供数据源建议、搜索关键词、筛选标准
- **资格评估（Qualification）**：BANT / MEDDIC / SPICED 三种评估框架
- **客户评分（Scoring）**：Fit（匹配度）+ Intent（意向度）+ Timing（时机）三维评分矩阵
- **外展策略**：基于评分结果的差异化接触策略

## 使用示例

**示例 1：自动洗车机 ICP 定义**
```
/挖掘客户
行业：加油站运营
产品名称：SmartWash X9 全自动洗车机
地区：华东地区（江浙沪皖）
```

**示例 2：SaaS 客户挖掘**
```
/prospect
行业：汽车后市场服务
产品名称：CloudCarWash SaaS
ICP描述：拥有5-30家门店的连锁洗车品牌，目前使用Excel或纸质记录管理，年营收500万以上
```

## 执行流程

### 第一步：ICP 画像构建
基于行业模板或用户自定义描述，构建五维客户画像：行业属性、企业规模、技术现状、核心痛点、决策链结构。

### 第二步：数据源定位
识别目标客户的聚集地：行业协会名录、招聘网站分析、工商注册数据、地图 POI 数据、展会参展商名单等。

### 第三步：资格评估框架
选择合适的评估框架（推荐 BANT 用于标准产品销售，MEDDIC 用于复杂方案销售），为每个维度定义评分标准。

### 第四步：客户评分矩阵
生成 Fit-Intent-Timing 三维评分表，自动计算综合得分并划分优先级（Hot / Warm / Cold）。

### 第五步：差异化外展策略
对 Hot 级客户（高匹配+高意向+好时机）设计精准外展序列，对 Warm 和 Cold 级客户分别设计培育和长期孵化方案。

```markdown
# SmartWash X9 目标客户挖掘 — 华东地区加油站

## 🎯 理想客户画像（ICP）

| 维度 | 标准 |
|------|------|
| 🏭 行业 | 民营加油站 / 加盟连锁加油站 |
| 📏 规模 | 日均进站车辆 200+ 台，至少 2 条加油通道 |
| 💻 技术现状 | 无自助洗车设备，或使用老旧设备（>5年） |
| 😫 核心痛点 | 洗车排队流失客户、人工洗车成本高、场地利用率低 |
| 👥 决策链 | 加油站老板/总经理（最终决策）→ 运营经理（评估）→ 财务（审核） |

## 📋 数据源清单

| 序号 | 数据源 | 可获取信息 | 优先级 |
|------|--------|-----------|--------|
| 1 | 中国加油站协会会员名录 | 企业名称、地址、规模 | ⭐⭐⭐ |
| 2 | 高德/百度地图 POI | 加油站位置、联系方式 | ⭐⭐⭐ |
| 3 | 天眼查/企查查 | 企业注册信息、股东 | ⭐⭐ |
| 4 | 加油站行业展会参展名单 | 决策人联系方式 | ⭐⭐ |
| 5 | 加油站设备采购招标公告 | 采购意向、预算 | ⭐⭐⭐ |

## 📊 客户评分矩阵

| 评分维度 | 权重 | 评分标准 |
|----------|------|----------|
| 🎯 匹配度 | 40% | 企业规模 + 技术现状 + 场地条件 |
| 🔍 意向度 | 35% | 近期搜索行为 + 招标信息 + 竞品询价 |
| ⏰ 时机 | 25% | 设备更新周期 + 扩张计划 + 预算周期 |

### 评分结果示例
| 客户名称 | 匹配度 | 意向度 | 时机 | 总分 | 等级 |
|----------|--------|--------|------|------|------|
| 杭州顺达加油站 | 38/40 | 30/35 | 20/25 | 88 | 🔥 Hot |
| 南京通达石化 | 35/40 | 25/35 | 15/25 | 75 | 🟡 Warm |
| 合肥路安加油站 | 30/40 | 18/35 | 10/25 | 58 | 🔵 Cold |

## 📩 外展策略

### 🔥 Hot（>80 分）— 立即接触
- Day 1：电话联系运营经理，直接约面谈
- Day 3：发送案例视频 + ROI 计算器
- Day 7：邀请参观已安装站点

### 🟡 Warm（65-80 分）— 积极培育
- 加入行业资讯邮件列表
- 邀请参加线上产品演示会
- 定期发送行业白皮书和案例

### 🔵 Cold（<65 分）— 长期孵化
- 品牌曝光（公众号关注、SEO 内容触达）
- 行业活动邀请
- 季度性触达邮件
```
