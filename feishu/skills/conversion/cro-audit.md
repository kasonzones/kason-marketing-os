# 🧪 转化率优化审计 (CRO Audit)

对营销着陆页、产品详情页进行全面转化率诊断，以数据驱动的方式找出阻碍转化的关键问题并给出优化方案。

## 7 维度审计框架

| 维度 | 检查要点 |
|------|----------|
| 🎯 **价值主张清晰度** | 用户3秒内是否理解产品做什么、为谁做、为什么选你？ |
| 📰 **标题有效性** | 主标题是否直接回应目标用户的痛点？副标题是否展开价值？ |
| 🖱️ **CTA 行动号召** | 按钮文案是否具体（"获取报价" vs "提交"）？位置、颜色、数量是否合理？ |
| 👁️ **视觉层级** | 重要元素是否突出？视线扫描路径是否符合 F/Z 型？ |
| 🛡️ **信任信号** | 是否有案例/评价/认证/保障？社会证明是否可见？ |
| ⚔️ **异议处理** | 价格、隐私、效果疑虑是否有针对性回答？ |
| ⚡ **摩擦点** | 表单字段是否过多？加载速度？移动端体验？ |

## 输出格式

```markdown
## 🔍 CRO 审计报告 — [页面名称]

### ✅ 审计概况
- 审计时间：YYYY-MM-DD
- 当前预估转化率：X%
- 核心问题：一句话总结

### 🟢 Quick Wins（立即执行，0-2天）
1. **问题描述** → **修复方案** → **预估提升**
2. ...

### 🟡 High-Impact Changes（高影响变更，3-7天）
1. **问题描述** → **修复方案** → **预估提升**
2. ...

### 🔵 Test Ideas（A/B测试方案）
1. **假设** → **变量** → **成功指标**
2. ...
```

## AutoWash Pro X1 示例

### 场景：AutoWash Pro X1 产品着陆页审计

```markdown
## 🔍 CRO 审计报告 — AutoWash Pro X1 产品页

### ✅ 审计概况
- 审计时间：2026-07-13
- 当前预估转化率：1.8%（行业基准 3.2%）
- 核心问题：价值主张埋没在技术参数中，CTA分散且缺乏紧迫感

### 🟢 Quick Wins
1. **首屏无CTA按钮** → 在Hero区域添加"获取报价"浮动按钮 → +0.5%
2. **移动端表单10个字段** → 精简为"姓名+电话+站点类型"三步 → +0.3%
3. **缺少真实客户案例** → 首屏下方添加3个加油站/酒店客户的Before/After卡片 → +0.4%

### 🟡 High-Impact Changes
1. **标题讲产品不讲结果** → "全自动洗车机"改为"每天多洗80辆车，人力成本降60%" → +1.2%
2. **无投资回报计算器** → 添加ROI计算工具，输入日均洗车量即刻显示回本周期 → +0.8%
3. **信任信号在页面底部** → 将"已服务500+站点"徽章移至首屏 → +0.3%

### 🔵 Test Ideas
1. **假设**：价格透明 vs 价格隐藏 → 变量：是否在首屏显示起售价 → 指标：询盘率
2. **假设**：视频演示 vs 图片轮播 → 变量：首屏媒体类型 → 指标：停留时长+转化率
3. **假设**：即时咨询 vs 表单留言 → 变量：CTA触发方式(在线客服/表单) → 指标：有效线索量
```

## 执行流程

1. 📋 **收集页面信息** — URL、流量数据、现有转化率、用户行为录屏
2. 🔬 **7维度逐一审计** — 每个维度打分（1-5）并记录发现
3. 📊 **输出审计报告** — 按Quick Wins → High Impact → Test Ideas分类
4. 🎯 **优先级排序** — 按预估提升 × 实现难度排序
5. 🚀 **跟踪验证** — 记录每次变更后的转化率变化
