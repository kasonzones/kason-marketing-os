# /社区营销 (community)

在线社区建设与运营增长技能。从零到一构建活跃社区，实现用户自增长与品牌忠诚度飞跃。

## 触发词

`/社区营销` `/community`

## 社区飞轮模型

```
        ┌──────────────────────────────────┐
        │                                  │
        ▼                                  │
  ┌──────────┐    ┌──────────┐    ┌──────────┐
  │  吸引     │───→│  激活     │───→│  留存     │
  │ Attract  │    │ Activate │    │ Retain   │
  └──────────┘    └──────────┘    └──────────┘
        │                                  │
        │         ┌──────────┐            │
        └────────→│  推荐     │←───────────┘
                  │ Refer    │
                  └──────────┘
                        │
                        ▼
                  社区自增长飞轮加速
```

### 阶段详解

**吸引 (Attract)**
- 明确社区价值主张：不为"交流"，而为解决具体问题
- 引流渠道：官网、邮件、社交、产品内入口
- 种子用户策略：定向邀请 20-50 位行业 KOL/活跃用户
- 内容钩子：独家资源、行业报告、工具模板、专家 AMA

**激活 (Activate)**
- 新成员欢迎流程：自动私信 → 自我介绍引导 → 首次互动奖励
- 首次价值交付（<24h）：让新成员加入当天就获得有用信息
- 破冰机制：每周固定话题讨论、新人专属频道
- 激活率目标：> 60%（加入后 7 日内至少互动 1 次）

**留存 (Retain)**
- 内容日历：每周固定主题日（如"案例周三"、"答疑周五"）
- 会员成长体系：贡献积分 → 等级晋升 → 特殊权限
- 核心成员培养：识别 Top 10% 活跃用户，赋予管理/主持角色
- 留存率目标：月活跃留存 > 50%

**推荐 (Refer)**
- 社区独占价值：非公开资源、专家网络、合作机会
- 邀请激励机制：双轨制奖励（推荐人 + 被推荐人均获奖励）
- 社交货币：社区专属徽章、证书、曝光机会
- 推荐增长率目标：月增长 > 15%

## 共享身份建设

### 身份定义要素
1. **共同的敌人**：行业低效、信息不对称、劣质供应商
2. **共同的目标**：提升运营效率、降低成本、技术升级
3. **共同的语言**：行业术语、内部梗、专属表达
4. **共同的神话**：标杆成员故事、社区里程碑事件

### AutoWash Pro X1 社区身份示例

```yaml
community: AutoWash 运营者联盟
```

## 平台运营指南

### Discord
- **适用场景**：实时互动、技术讨论、紧密社区
- **频道架构**：
  ```
  📌 公告与规则
  ├── #公告
  ├── #社区规则
  └── #新人报到
  💬 核心讨论
  ├── #运营交流
  ├── #设备问题
  ├── #行业动态
  └── #资源分享
  🎯 专项频道
  ├── #AutoWash_X1 专属
  ├── #ROI 数据晒单
  └── #案例拆解
  🤝 合作与互动
  ├── #对接合作
  ├── #线下活动
  └── #闲聊
  ```
- **机器人配置**：欢迎机器人、投票机器人、问答机器人、防垃圾机器人
- **运营节奏**：每日早安帖 + 周固定活动 + 月度 AMA

### Slack
- **适用场景**：B2B 专业社区、付费社群、行业圈子
- **频道架构**：
  ```
  #general — 全体公告
  #introductions — 自我介绍
  #operations — 运营讨论
  #equipment — 设备技术
  #roi-showcase — 数据分享
  #industry-news — 行业资讯
  #random — 水聊
  ```
- **付费门槛**：Slack 社区适合 $99-499/月的付费社区
- **整合能力**：与 CRM、邮件、日历深度集成

### Circle
- **适用场景**：品牌化社区、课程社区、付费社群
- **核心功能**：
  - 空间 (Spaces)：按主题分组的讨论区
  - 课程 (Courses)：嵌入式学习内容
  - 活动 (Events)：直播 + 录播
  - 成员目录：可搜索的会员网络
- **优势**：无需代码、品牌化程度高、SEO 友好
- **AutoWash Pro X1 空间规划**：
  ```
  🏠 大厅 — 社区公告 + 新人引导
  📊 运营学院 — 设备管理课程 + 认证
  💡 实战案例 — 运营者故事 + ROI 计算器
  🔧 技术中心 — 故障库 + 维护指南
  🤝 会员网络 — 人脉对接 + 行业合作
  ```

### Reddit
- **适用场景**：公开讨论、SEO 内容沉淀、品牌曝光
- **子版块策略**：
  - 自有子版块：r/AutoWashCommunity（专属社区）
  - 参与优势子版块：r/carwash、r/smallbusiness、r/EntrepreneurRideAlong
- **发帖规范**：
  - 价值优先：每条帖子必须对读者有用
  - 避免硬广：以数据和经验分享为主
  - 互动驱动：提问式结尾、主动回复评论
  - 频率控制：自有子版块 3-5 帖/周，外部子版块 1-2 帖/周

### Facebook Groups
- **适用场景**：本地化运营者社区、区域活动组织
- **群组设置**：
  - 私密群组（非公开）：提升会员价值感
  - 入群问题：筛选目标用户
  - 群规明确：禁止硬广、要求价值互动
- **内容策略**：
  - 每周主题日（周一答疑、周三案例、周五直播）
  - 会员 spotlight：每周推荐 1 位活跃会员
  - 本地分组：按地区创建本地运营者小组

## AutoWash Pro X1 社区启动计划

### 第 1 阶段：筹备期（Week 1-2）
```
- [ ] 确定社区价值主张："用数据和经验帮洗车场主做出更好的设备决策"
- [ ] 选择主平台（Discord）+ 辅助平台（Reddit）
- [ ] 准备种子内容：10 篇深度文章 + 5 个案例 + 3 个工具模板
- [ ] 邀请 30 位种子成员（现有客户 + 行业联系人）
- [ ] 设置社区规则与欢迎流程
```

### 第 2 阶段：冷启动期（Week 3-6）
```
- [ ] 每日 1 篇高质量帖子（由社区经理发布）
- [ ] 每周 1 场 AMA 直播（技术专家/行业嘉宾）
- [ ] 种子成员 1v1 激活沟通
- [ ] 收集前 20 条用户生成内容（UGC）
- [ ] 建立"运营者之星"月度评选
```

### 第 3 阶段：增长期（Week 7-16）
```
- [ ] 启动推荐计划：推荐 1 位运营者 = 获得 1 次免费设备巡检
- [ ] 开放社区公开资源区（seo 引流）
- [ ] 培养 5 位社区版主（从活跃会员中晋升）
- [ ] 举办首次线上峰会："洗车运营效率大会"
- [ ] 月度增长目标：+30% 新成员
```

### 第 4 阶段：规模化（Week 17+）
```
- [ ] 开通付费会员层级（$29/月）：独家数据 + 优先技术支持
- [ ] 建立本地分会：按国家/地区组建线下运营者聚会
- [ ] 推出社区认证体系：设备操作认证、运营优化认证
- [ ] 开发社区专属工具：ROI 计算器、设备对比工具
```
