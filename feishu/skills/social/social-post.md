# /社交媒体 (social-post)

社交媒体内容创作技能，覆盖 LinkedIn、Twitter/X、Instagram、TikTok 四大平台的内容策划与生成。提供从内容支柱到发布模板的完整工作流。

## 触发词

`/社交内容` `/social-post`

## 内容支柱体系

### 支柱 1：教育内容 (Educational)
- 占比：40%
- 目标：建立行业权威，提供实用价值
- 形式：操作指南、数据分析、行业洞察、常见误区
- 平台适配：
  - LinkedIn：长文 + 数据可视化
  - Twitter/X：线索推文（Thread）
  - Instagram：轮播图 + 信息图
  - TikTok：15-30s 教学短视频

### 支柱 2：社交证明 (Social Proof)
- 占比：25%
- 目标：建立信任，展示真实效果
- 形式：客户案例、使用前后对比、用户反馈截图、测评数据
- 平台适配：
  - LinkedIn：案例研究深度文章
  - Twitter/X：快速结果 + 截图
  - Instagram：前后对比轮播
  - TikTok：客户访谈/UGC 混剪

### 支柱 3：品牌故事 (Brand Story)
- 占比：20%
- 目标：建立情感连接，传递品牌价值观
- 形式：创业历程、团队故事、用户故事、公益行动
- 平台适配：
  - LinkedIn：创始人深度文章
  - Twitter/X：幕后花絮 + 短故事
  - Instagram：团队日常 + 用户故事
  - TikTok：公司文化 + 趣味花絮

### 支柱 4：互动/娱乐 (Engagement/Entertainment)
- 占比：15%
- 目标：扩大触达，提升互动率
- 形式：投票、问答、挑战、梗图、行业趣事
- 平台适配：
  - 各平台原生互动功能优先

## 标题/钩子公式

### LinkedIn 钩子公式
1. **数据震撼**："{数字}% 的 {目标受众} 还在 {做错误的事}"
2. **反常识**："为什么 {公认的做法} 其实在伤害你的 {业务指标}"
3. **痛点共鸣**："如果你正在为 {痛点} 头疼，这篇给你答案"
4. **结果预告**："我们帮 {客户类型} 把 {指标} 从 {A} 提升到 {B}，方法如下"

### Twitter/X 首推公式
1. **直接陈述**："{核心观点}。"
2. **对比钩子**："{做法 A} vs {做法 B}，差距比你想的大"
3. **列举式**："{N} 个 {主题} 的真相"
4. **提问式**："{目标受众}，你们遇到 {问题} 的时候是怎么处理的？"

### Instagram 标题公式
1. **双读+CTA**："你以为...实际上.../ 评论区告诉我你的想法"
2. **列表预览**："{N} 种 {做法}，你试过几种？/ 左滑看全部"
3. **前后对比**："第 1 张 vs 第 3 张，差别在哪里？"

### TikTok 开场公式
1. **模式中断**：前 0.5 秒视觉冲击 → 快速切入主题
2. **好奇缺口**："{惊人结果} — 但方法不是你想的那样"
3. **快速价值**："{N} 秒教你 {做某事}"

## 内容再利用系统

```
一篇深度文章
  ├── LinkedIn 主帖（精简到 1200 字）
  ├── Twitter/X 线索（拆成 5-8 条推文）
  ├── Instagram 轮播（拆成 8-10 页图文）
  ├── TikTok 短视频（提炼 1 个核心观点，45 秒内）
  └── Newsletter 邮件（扩展到 2000 字 + 独家洞察）
```

## 轮播图框架

### 框架 1：问题-解决型（8-10 页）
1. 封面：吸引眼球的标题
2. 问题陈述：让受众点头
3. 普遍做法：常见的（错误）方式
4. 为什么不行：2-3 个原因
5. 正确做法：策略核心
6. 步骤 1：具体操作
7. 步骤 2：具体操作
8. 步骤 3：具体操作
9. 预期结果：量化收益
10. CTA + 关注引导

### 框架 2：数据洞察型（6-8 页）
1. 封面：核心数据点
2. 数据来源/背景
3. 发现 1：数据 + 解读
4. 发现 2：数据 + 解读
5. 发现 3：数据 + 解读
6. 趋势判断
7. 行动建议
8. CTA + 关注引导
